# Rumikub
Rumikub simulator in c++.

Project done in FP2 during my first year at university.
Consists of a simulation of the card game "Rumikub", where you have to get rid of all your cards (grouping them with the ones on your hand or the ones on the table by same number diferent color or making color straights).
